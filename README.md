# Lucator - Lua Code Obfuscator

Lucator is a powerful yet straightforward Lua code obfuscator designed to protect your Lua scripts while maintaining functionality. The name "Lucator" combines "Lua" and "Obfuscator" to create a unique identity for this tool.

## Features

- :lock: String Encryption
- :arrows_counterclockwise: Variable Renaming
- :cyclone: Control Flow Obfuscation
- :package: Table Structure Shuffling
- :broom: Comment & Whitespace Removal
- :zap: Optimized Performance
- :shield: Anti-Tamper Protection

## Requirements

- Lua 5.1 - 5.4
- LuaJIT (optional)
